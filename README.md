# Min

Vue.js starter App


## Install

```console
npm install
```


## Run

```console
npm run dev
npm run build
```


## Lint

Lint .js & .vue files
```console
npm run eslint
npm run eslint:fix
```

Lint .css, .scss, .sass & .vue files
```console
npm run stylelint
npm run stylelint:fix
```

Do both
```console
npm run lint
npm run lint:fix
```
